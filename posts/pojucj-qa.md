## 五、方法框架（道法术器）

### 道（底层认知）

#### 1. 大众崇拜两种 IP（你必须占其一）
- 站在高位还愿意帮助普通人的
- 吃过苦经历过跌宕、走出困境后继续发展的

**没有高位 → 必须打"反差"**。这是普通人 IP 的唯一护城河。

#### 2. 不要从能力顺推，要从目标倒推
> "布局节奏要从最终目标倒推——而不是从现有能力出发做规划。"

顺推 → 容易被现有能力锁住；倒推 → 自动跨越能力差距。

#### 3. 大钱在细分痛点，不在通用方向
> "**真正的大钱是在要么是细分行业的绝对痛点**——只解决行业内的一个问题，就可以撬动年度增长。"

通用方向被大公司抢光——细分痛点才是普通人的活路。

#### 4. 早就是优势
> "未来所有行业都需要去加 AI——就像互联网时代一样。**当所有行业都必要用 AI 赋能的时候，那个时候就是万物 AI——早就是优势。**"

#### 5. 选择圈子比单点能力更重要
> "**AI 时代选择一个好的圈子是非常重要的**——圈子带给你的价值不仅是就业机会，还有商业思维的全方位提升。"

> "圈子带给你的价值——不仅仅是就业机会，还有商业思维的全方位提升。"

#### 6. 什么拯救了你，就用什么回报世界
安娜的金句——这是创业者的内驱力底层：**找到你被拯救的那件事 → 让更多人也被它拯救 → 这就是你的事业**。

#### 7. 破局的投资是"感应机制"
> "破局的投资是会员回馈的感应机制——不会提前布局特定赛道。"

> "**会员在社群内完成 0 到 1、0 到 10 的成长——证明了自己的领导力与执行力——破局就会在可承受范围内提供投资支持。**"

意思是：**先证明自己 → 再有资金**（不是相反）。

#### 8. 学习 AI 不需要懂技术
> "如果你的目标是用 AI 改造现有行业——**那只需要学会 AI 提问、用好 AI 工具，就已经超过 99% 的从业者**。"

### 法（方法论）

#### 法 1：传统资源 AI 变现三层路径

```
第一层：打造个人 IP 面向企业老板
   ↓ 让客户知道你会用 AI
   ↓ 完成体验转化

第二层：在服务过程中组建自己的团队
   ↓ 把 IP 变成可交付的服务

第三层：针对身边老板的行业资源做细分
   ↓ 聚焦细分行业的刚需痛点
   ↓ 容易拿到大结果
```

> "细分痛点更容易拿到大结果——通用方向更容易被大公司抢占。"

#### 法 2：童装/B 端业务 AI 增长 3 步漏斗

| 步骤 | 动作 | 是否需 AI |
| --- | --- | --- |
| **1. 老客户转介绍** | 介绍成功的团长 20% 分成 | ✗ 不需要 |
| **2. 个人 IP 内容** | 公众号 / 微博 / 视频号发布 | ✓ AI 提升内容产能 |
| **3. 投放放大** | 业务规模扩大后再投流 | ✓ 视情况 |

#### 法 3：低谷反差 IP 打造 3 步法

```
1. 找你过往最真实的低谷故事
   → 不是简历——是有人物有冲突的"经历档案"

2. 把低谷故事 与 现在的梦想形成反差
   → 反差越大，信任越强

3. 持续在内容里讲反差
   → IP 自然成型
```

#### 法 4：多方向布局的目标倒推法

```
错误做法（顺推）：
   "我会 X、Y、Z" → "所以我同时做 X、Y、Z"
   → 分散精力 → 没结果

正确做法（倒推）：
   "我的最终目标是用 AI 改造 X 行业"
   → "所以我只需要学会 AI 提问 + 应用"
   → "已经超过 99% 从业者"
   → 聚焦做深 X
```

#### 法 5：大学生时间有限的 4 步路径

```
1. 优先聚焦流量（找最高效率获客平台）
   ↓ 投入主要精力
2. 不急着做产品（流量不够前不做）
   ↓
3. 私域积累（慢慢沉淀关系）
   ↓
4. 推荐别人产品分佣（破局最高 40%）
   → 满足初期收入需求
```

#### 法 6：普通人多赛道选择 3 原则

在多个方向里只选 1 个：
- 最有优势 / 最喜欢 / 最容易赚钱

> "不要分散精力同时做多条线。"

#### 法 7：破局的投资感应机制

```
会员先在社群内完成 0→1 / 0→10 成长
   ↓ 证明领导力 + 执行力
有创业需求时
   ↓
破局在可承受范围内提供投资支持
（不是先看赛道，是先看人）
```

1. 高门槛筛选同频人（破局的会员费就是筛选）
2. 最前沿的信息流通
3. 大量线下深度链接（每年 8-10 场大型活动）
4. 带教练带学员的传承链条

#### 术 4：副业先做到本职持平再 all in
- 先副业验证
- AI 收入做到本职持平
- 再考虑全职转型

破局当前 ¥1,xxx → 6.13 起 ¥2,399。**直播间专属优惠券 + 7 天无理由退换**。**今天买 = 锚定低价**。

#### 术 6：拼豆模式 = 找有相同基因的合伙人
翠姐的拼豆所有合伙人都在破局找到——**社群不只是学习场，是合伙人筛选器**。

#### 术 7：用"我做过哪三件让自己骄傲的事"开始 IP
不要从"我有什么能力"开始——从**最戳到自己的故事**开始。低谷故事 + 反差 + 当前目标 = IP 的种子。

**学阿里前员工那条铁律：从目标倒推，不要从能力顺推**。具体动作：
1. 写下你最终想成为什么样
2. 倒推：要达到这个目标，最关键的 1 件事是什么？
3. 砍掉其他所有方向，只做这 1 件

### 如果你有传统资源但不会 AI 变现
**传统软件那个提问者就是你**——300+ 企业、几千人脉、政府关系。**三层路径**：先打 IP（让客户知道你会 AI）→ 服务中组建团队 → 聚焦细分行业刚需痛点。**通用方向被大公司吃光，细分痛点是普通人的活路**。

### 如果你担心"我光环不够、资源不够、能力不够"
