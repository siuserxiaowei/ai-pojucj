## 四、核心故事

最戳人的部分是**编程时代划分**——

| 时代 | 模式 | 核心动作 |
| --- | --- | --- |
| L0 | ChatGPT 出现前 | 手抄代码 |
| L1 | Copilot 时代 | 自动补全 |
| L2 | Web Coding 时代 | 人机协作 |
| **L3** | **当前** | **管理多个 Agent 自主运行，人只盯节点检查问题** |

第二个戳人的是**程序员误区警告**：

> "部分程序员沉迷学习但止于学习——喜欢从底层源码、计算机原理开始逐步学习，陷入学无止境反而丢失自身方向的误区。"

意思是：**L3 时代的核心能力不是写代码，是描述清楚需求 + 调试 Agent。**

## 五、方法框架（道法术器）

### 道（底层认知）

- **L3 时代核心红利窗口期 2 年**：2027 年就会非常内卷，2026 年是入场最后机会
- **当前已进入编程领域 AGI 时代**：顶级模型差距只在细枝末节
- **种树逻辑 vs 摘果子逻辑**：AI 编程出海是种树，半年到一年才见结果，但上限很高
- **少即是多**：找准唯一的定位，避免同时深入多方向
- **ESBI 象限的杠杆使用**：L3 时代普通人能用劳动力杠杆 + 编程杠杆 + 资本杠杆

### 法（方法论）

#### 1. 时代划分判别法

| 你在哪个时代 | 表现 |
| --- | --- |
| L0 | 还在手抄、复制粘贴代码 |
| L1 | 用 Copilot 自动补全 |
| L2 | 用 Cursor / Claude Code 协作开发 |
| **L3** | **管理多个 Agent，自己只盯节点** |

如果你还在 L1-L2，立刻升级到 L3，这是入场红利的核心要求。

#### 2. AI 编程做产品的核心能力

| 能力 | 重要性 |
| --- | --- |
| 写代码 | ↓ 已经不重要 |
| 描述需求 | ↑ 决定上限 |
| PRD 设计 | ↑ 直接决定产品质量 |
| 调试 Agent | ↑ 未来 2 年核心竞争力 |
| 业务理解深度 | ↑↑ AI 解决广度，人提供深度 |

#### 3. 关键词选择思路

- **找新词**：依托趋势快速成长，适合前期练手
- **长期布局**：高 LTV 的稳定需求，搏概率
- 两手都要：新词练手，老词养老

#### 4. 时间结构重构

- 个人自然时间 = 24 小时不变
- L3 时代 Agent 7×24 运行
- 利用非高峰期（避开 token 高峰），成本更低
- 搭建带组织架构的 Agent 系统 → 增长是指数级

### 术（具体策略）

#### 程序员入门 AI 编程出海三步

1. **快速浏览所有训练营文档**（一周内）
2. **找准唯一定位**：少即是多
3. **避开"学无止境"陷阱**：直接做项目，边做边学

#### PRD 设计核心要素

- 价值主张
- 差异化竞争优势
- 竞品分析
- 成本核算
- 定价

PRD 质量 = 产品质量上限。

### 器（工具）

- **GPT5.5 / JM5.1 / DeepSeek V4**（顶级模型）
- **Cursor / Claude Code**（L2 级协作工具）
- **多 Agent 框架**（L3 级自治系统）
- **PRD 模板库**（决定产品上限）
